# 投射呪法訓練場

「投射呪法」の「3秒予約→1秒実行」サイクルを再現したブラウザベースのランアクションゲームです。
ウェブカメラを用いたリアルタイム骨格検出（エッジAI）による直感的なジェスチャー操作と、キーボード操作の両方に対応しています。

## 🎮 ゲームの概要

プレイヤーは迫りくる障害物を回避しながら、より長い距離を走破することを目指します。

### ２つのゲームモード
1. **投射呪法モード**
   - 3秒間（または5秒間）の「予約フェーズ」で自分の行動（移動、ジャンプなど）を事前に入力します。
   - その後、「発動フェーズ」で予約した行動が自動・高速で実行され、障害物を突破します。
2. **リアルタイムモード**
   - 予約なしで、一般的なランゲームのようにリアルタイムに迫りくる障害物を避けていきます。

### 障害物と対応アクション
- **壁**: レーンを移動して避ける
- **茨 (接地)**: 跳ぶ (Jump)
- **天 (上部)**: 屈む (Crouch)
- **窟 (極低空)**: 伏せる (Pushup)

## 🕹️ 操作方法

### カメラ操作（推奨）
PCのウェブカメラを使用し、プレイヤーの実際の体の動きをAIが検出してキャラクターを操作します。

- **レーン移動**: 体を左右に動かす
- **跳ぶ (Jump)**: ジャンプする、または腰を上げる
- **屈む (Crouch)**: しゃがむ、または腰を下げる
- **伏せ (Pushup)**: 深くしゃがむ、または床に伏せる

### キーボード操作
カメラが無い環境でもプレイ可能です（設定パネルでカメラ入力を「無効」にしてください）。

- **レーン移動**: `←` / `→` キー
- **跳ぶ (Jump)**: `Space` / `↑` キー
- **屈む (Crouch)**: `Shift` / `↓` キー
- **伏せ (Pushup)**: `Z` / `C` キー

## 🛠️ 技術スタック

当プロジェクトは、サーバーサイド通信を一切行わない完全なクライアントサイド（ブラウザ）完結型のアプリケーションとして構築されています。

### フロントエンド (基盤)
- **言語**: TypeScript v5.7
- **ビルド**: Vite v6.0
- **スタイル**: CSS3 (Variables, Animations)

### グラフィックス / 演出
- **レンダリング**: Canvas API (Custom Renderer)
- **エフェクト**: Procedural Generation（プロシージャル生成による波紋、墨、パーティクル表現）

### AI / ML (エッジコンピューティング)
- **骨格検出・ジェスチャー判定**: MediaPipe Pose (`@mediapipe/pose`)
  - ユーザーのデバイス上でリアルタイムに推論エッジ処理を実行し、プライバシーと低遅延を両立。

### インフラ / その他
- **環境**: ブラウザ (Client-side only, No Backend)
- **オーディオ**: Web Audio API（BGM / SE ロジック）

## 🚀 ローカル環境の構築と実行

Node.js (v18以降推奨) がインストールされている環境で、以下のコマンドを実行してください。

```bash
# パッケージのインストール
npm install

# 開発サーバーの起動 (localhost:5173 などでアクセス可能になります)
npm run dev
```

### ビルド
本番環境向けの静的ファイルを生成する場合は以下のコマンドを実行します。
```bash
npm run build
```
生成されたファイルは `dist/` ディレクトリに出力されます。
